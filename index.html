Mengenal Website, HTML, dan Visual Studio Code

Definisi Website

Situs web adalah kumpulan halaman web dan konten terkait yang diidentifikasi dengan nama domain umum dan dipublikasikan di setidaknya satu server web. Ini dapat berisi teks, gambar, video, dan aset digital lainnya yang dapat diakses melalui internet.

Definisi HTML

HTML adalah bahasa markup yang digunakan untuk membuat dan menyusun halaman web. Tanpa HTML, tampilan web akan berantakan dan tidak terstruktur dengan baik.

Kekurangan Website dengan HTML

  • HTML biasa digunakan untuk membuat halaman web statis, sehingga jika ingin fitur yang dinamis perlu menggunakan bahasa backend seperti PHP atau JavaScript.
  • Tidak memungkinkan pengguna untuk menjalankan logika.
  • Fitur baru tidak dapat dijalankan secara cepat di sebagian browser.

Kelebihan Website dengan HTML

  • Bersifat open-source serta sepenuhnya gratis.
  • Memiliki bahasa markup yang rapi dan konsisten.
  • Memiliki banyak sumber dan komunitas yang besar.
  • Mudah diintegrasikan dengan bahasa backend.
  • Dijalankan di setiap web browser.

Manfaat HTML

Selain sebagai branding dan profil perusahaan, website juga bermanfaat sebagai sumber informasi dan memudahkan komunikasi. Dengan memiliki website, Anda bisa dengan mudah memperbarui informasi bagi pengguna situs Anda secara cepat dan rutin.

Langkah-langkah Membuat HTML Menggunakan VS Code

Langkah Membuat File

  1. Unduh aplikasi VS Code di sini.
  2. Setelah terunduh dan terinstal di PC Anda, unduh Live Server dalam aplikasi VS Code (opsional) untuk memudahkan penyimpanan.
  3. Buat folder baru di komputer/laptop Anda dengan nama "HTML".
  4. Buka aplikasi VS Code dan buat file baru di dalam folder tersebut dengan nama "index.html".

Template HTML

Template HTML adalah kerangka dasar atau struktur HTML yang digunakan sebagai dasar untuk membuat halaman web dengan desain atau tata letak yang serupa. Template ini membantu mempercepat proses pengembangan web dengan menyediakan elemen-elemen dasar yang dapat diisi atau dimodifikasi sesuai kebutuhan. Template HTML biasanya mencakup bagian-bagian seperti header, footer, sidebar, dan area konten utama, yang bisa diisi dengan konten dinamis atau statis.

Penjelasan Elemen:

  • <!DOCTYPE html>: Memberitahu browser bahwa dokumen menggunakan HTML versi terbaru.
  • <html lang="en">: Menentukan bahasa dokumen sebagai bahasa Inggris.
  • <meta charset="UTF-8">: Menentukan karakter encoding sebagai UTF-8.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Menentukan tampilan yang responsif sesuai dengan ukuran layar perangkat.
  • <title>Document</title>: Mengubah judul website yang akan terlihat pada tab browser.

Elemen-elemen ini biasanya tidak perlu diubah kecuali dibutuhkan secara khusus.

Heading

Heading dalam HTML digunakan untuk menentukan judul atau subjudul pada sebuah halaman web. HTML menyediakan enam tingkat heading, dari <h1> hingga <h6>, di mana <h1> adalah heading dengan tingkat tertinggi dan <h6> dengan tingkat terendah.

Paragraf

Paragraf dalam HTML didefinisikan dengan tag <p>. Elemen ini digunakan untuk menyusun teks agar mudah dibaca dan diatur secara logis.

List

Daftar dalam HTML digunakan untuk menyusun item-item menjadi bentuk yang terstruktur. Ada dua jenis daftar utama:

  • Ordered List (<ol>): Daftar berurut dengan tipe angka, huruf kapital, huruf kecil, atau angka Romawi.
  • Unordered List (<ul>): Daftar tidak berurut dengan tipe square, disk, atau none.

List Item (<li>): Elemen yang digunakan untuk mendefinisikan item dalam daftar.

Anchor

Anchor (<a>) dalam HTML digunakan untuk membuat tautan atau hyperlink ke halaman web lain atau ke berkas lainnya. Atribut href untuk menentukan lokasi digunakan untuk menentukan URL tujuan.

Image

Elemen <img> digunakan untuk menampilkan gambar di halaman web. Elemen ini tidak memiliki tag penutup dan memiliki atribut.

Table

Elemen <table> dalam HTML digunakan untuk membuat tabel di dalam halaman web. Tabel terdiri dari elemen-elemen seperti:

  • <thead>: Kepala atau judul tabel.
  • <tbody>: Isi tabel.
  • <th>: Elemen untuk judul kolom (bold).
  • <td>: Elemen untuk data dalam tabel (normal).
  • <tr>: Elemen untuk baris dalam tabel.
  • <tfoot>: Kaki atau penutup tabel.

Form

Elemen <form> dalam HTML digunakan untuk membuat formulir di dalam halaman web. Formulir memungkinkan pengguna untuk mengirimkan data ke server untuk diproses.

Iframe

Elemen <iframe> dalam HTML digunakan untuk menanamkan sebuah halaman web atau konten dari situs web lain ke dalam halaman web. Atribut src menentukan URL dari halaman web yang akan ditampilkan di dalam iframe.

Dengan mengikuti langkah-langkah dan pemahaman di atas, Anda dapat membuat halaman web sederhana menggunakan HTML dan Visual Studio Code.
